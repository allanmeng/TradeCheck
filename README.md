# TradeCheck · 跨境交易网络合规自测工具

> 一个轻量、极简、现代暗黑科技风的单页自适应工具网页，用于检测券商交易链路的网络路由与合规状态。
# 访问这个地址：https://allanmeng.github.io/TradeCheck/
---

## 背景

随着跨境券商（富途牛牛、老虎证券、长桥证券等）对中国内地 IP 的交易风控收紧，内地交易者在普通代理模式下经常遭遇间歇性风控拦截。

券商的合规风控采取 **"最小化精准拦截"** 策略：**看盘（大盘行情）不限、平仓（卖出动作）不限，唯独精准锁死内地 IP 的【买入下单】动作**。

TradeCheck 的目标是让用户直观看到自己的分流规则是否 100% 免疫买入风控。

## 功能

### 🔍 本机 IP 查询
- **IPv4 / IPv6** — 多通道 IP 回显，国际/国内网络环境自适应
- **地理位置 & ISP & ASN** — 实时地理定位
- **分流对比** — ip138、Cloudflare、IPInfo 多源对比，直观看到软路由分流是否生效
- **连通性检测** — 微信、淘宝（国内直连）与 GitHub、YouTube（境外代理）的实时延迟与丢包点阵

### 📊 券商探测
基于 **三角定位法** 的三重交叉验证引擎：

| 技术 | 作用 |
|------|------|
| **DNS-over-HTTPS** | 解析券商子域名，获取 CDN 服务器 IP 及归属地 |
| **Image Ping** | 精确测量连通性与延迟，对比国内/国际基线 |
| **API 行为探测** | 模拟买入/卖出 API 请求，检测应用层拦截策略 |

覆盖三家券商、三条核心链路：

| 券商 | 📈 大盘 | 🛒 买入 | 💰 卖出 |
|------|---------|---------|---------|
| **富途证券** | 🌐 国际 CDN | 🚫 监管拦截机制 | ✅ 放行 |
| **老虎证券** | 🌐 国际 CDN | 🚫 监管拦截机制 | ✅ 放行 |
| **长桥证券** | 🌐 阿里云香港 | 🚫 监管拦截机制 | ✅ 放行 |

### 📋 内地监管政策
2026 年 6 月 12 日起，三家券商对中国内地账户的服务调整公告全文，含官方公告查询路径。

## 技术架构

纯前端单页应用，无需后端服务器：

```
浏览器 → IP Echo (ipify / ip.sb / IPIP)
       → DNS-over-HTTPS (Cloudflare / AliDNS / 腾讯DNS)
       → Image Ping (券商 CDN)
       → API Probe (fetch + no-cors)
```

所有检测均从浏览器直接发起，反映的是**用户本机网络栈的真实路由行为**。

## 使用方式

### 本地运行

```bash
cd TradeCheck
python -m http.server 8899 --bind 127.0.0.1
```

浏览器打开 `http://localhost:8899`。

### 一键启动（Windows）

直接双击 `start-server.bat`，关闭窗口即停止。

## 在线体验

部署到 GitHub Pages 或其他静态托管平台即可使用。
